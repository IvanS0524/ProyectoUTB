CSS

¿Qué es CSS?

CSS (Cascading Style Sheets) es el lenguaje utilizado para describir la presentación de un documento HTML. CSS controla el diseño visual y el estilo de las páginas web.

CSS es un lenguaje de diseño gráfico que se utiliza para dar estilo a las páginas web.

Se trata de la sigla que corresponde a Cascading Style Sheets, es decir, Hojas de Estilo en Cascada.

Caracteristicas

  • CSS es un lenguaje de diseño gráfico que se utiliza para dar estilo a las páginas web.
  • Selectores y Propiedades: CSS utiliza selectores para aplicar estilos a elementos específicos y propiedades para definir esos estilos.
  • Flexibilidad: CSS permite controlar el diseño y la presentación de una página web de forma precisa y detallada.
  • Reutilización: CSS permite reutilizar estilos en múltiples páginas, lo que facilita la consistencia y el mantenimiento del diseño.
  • Selectores y Propiedades: CSS utiliza selectores para aplicar estilos a elementos específicos y propiedades para definir esos estilos.

Sintaxis

  • Basado en reglas (selector { propiedad: valor; })..
  • Los selectores son los elementos a los que se les aplican los estilos.
  • Las propiedades son los estilos que se aplican a los elementos.
  • Los valores son los valores que toman las propiedades.
  • Se pueden aplicar estilos en línea, en el head o en un archivo externo.
  • Se pueden aplicar estilos a elementos, clases, id, etc.
  • Se puede usar en línea, interno o externo.

Ejemplo de Código en CSS


body {
  background-color: #121212;
  color: #ffffff;
  font-family: Arial, sans-serif;
  margin: 0;
  padding: 0;
}
