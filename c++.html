C++

C++ es un lenguaje de programación de propósito general, que se ha convertido en un lenguaje de programación muy popular.

Se utiliza ampliamente en el desarrollo de software de sistemas, aplicaciones de escritorio, videojuegos y más. Es conocido por su eficiencia y control sobre los recursos del sistema.

Caracteristicas

  • Programación Orientada a Objetos: C++ soporta la programación orientada a objetos, permitiendo la creación de clases y objetos.
  • Eficiencia y Rendimiento: C++ es conocido por su alto rendimiento y eficiencia en la gestión de recursos.
  • Bibliotecas Estándar: C++ cuenta con una amplia biblioteca estándar que facilita el desarrollo de aplicaciones complejas.

Sintaxis

C++ es un lenguaje compilado, lo que significa que el código fuente se traduce a código máquina antes de ejecutarse. La sintaxis de C++ es similar a la de C, pero incluye características adicionales como clases y objetos.

La sintaxis básica de C++ incluye:

  • Los programas en C++ comienzan con la función main().
  • Es necesario incluir la biblioteca iostream para utilizar cout, que es mostrar en pantalla.
  • A diferencia de Python, C++ requiere la declaración de variables antes de usarlas.
  • Al terminar una linea de codigo, se debe usar un punto y coma (;) para finalizarla.
  • Los condicionales funcionan de manera similar a otros lenguajes, utilizando if, else if y else, pero con una sintaxis más estricta.

Ejemplo de Código en C++

// Esto es un comentario
#include <iostream> //incluir biblioteca
using namespace std;
int main() { // función principal
    int x = 5; // declaración de variable
    int y = 10;
    if (x < y) { // condicional
        cout << "x es menor que y";
    } else {
        cout << "x es mayor o igual que y";
    }
    return 0;// fin de la función
}//fin del codigo
