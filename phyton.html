Phyton

Python es un lenguaje de programación de alto nivel, conocido por su simplicidad y legibilidad. Es ampliamente utilizado en desarrollo web, análisis de datos, inteligencia artificial y más.

Python es un lenguaje interpretado, lo que significa que no necesita ser compilado antes de ejecutarse.

Caracteristicas

  • Legibilidad: Python se destaca por su sintaxis clara y legible, que facilita la escritura y comprensión del código.
  • Sintaxis Clara y Concisa: Python tiene una sintaxis fácil de leer y escribir, lo que lo hace ideal para principiantes.
  • Comunidad activa: Python cuenta con una gran comunidad de desarrolladores y una amplia variedad de bibliotecas y frameworks que facilitan el desarrollo de aplicaciones.
  • Bibliotecas y Frameworks: Python cuenta con una gran cantidad de bibliotecas y frameworks que facilitan el desarrollo en diversas áreas.

Sintaxis

Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional.

  • "#" En Python se utiliza para agregar comentarios en una línea del código.
  • """ """ Se utiliza para agregar comentarios multilínea.
  • "print()" Se utiliza para mostrar información en la consola.

Ejemplo de Código en Python


        1 # Esto es un comentario
        2 print("Hola, mundo")
        3
        4 """
        5 Esto es un comentario
        6 multilínea
        7 """

Sintaxis

La sangría se refiere a los espacios al comienzo de una línea de código. En Python, la sangría es fundamental para definir bloques de código, como los que se encuentran dentro de funciones, bucles y condicionales.

La indentación es obligatoria en Python. Si no se utiliza correctamente, el código no funcionará.

La indentación puede ser de 4 espacios o un tabulador, pero es importante ser consistente en su uso.

  • En Python no es necesario declarar el tipo de variable
  • "if" y "else" son condicionales básicos y muy utilizados

Ejemplo de Código en Python


1 # Esto es un comentario
2 x = 5 #declaración de variable
3 y = 10
4 if x < y: # condicional
5     print("x es menor que y") #imprimir en consola
6 else:
7     print("x es mayor o igual que y")
