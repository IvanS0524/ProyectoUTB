HTML

¿Qué es HTML?

HTML (HyperText Markup Language) es el lenguaje estándar para crear y estructurar páginas web. Es el esqueleto de cualquier sitio web y define la estructura y el contenido de una página.

Caracteristicas

  • Elementos y Etiquetas: HTML utiliza etiquetas para definir elementos como encabezados, párrafos, enlaces, imágenes y más.
  • Atributos: Las etiquetas pueden tener atributos que proporcionan información adicional sobre los elementos.
  • Tablas: HTML permite crear tablas para organizar y mostrar datos de manera estructurada.
  • Formularios: HTML incluye elementos para crear formularios interactivos, como campos de texto, botones, casillas de verificación, etc.
  • Enlaces: HTML permite crear enlaces a otras páginas web, archivos, correos electrónicos, etc.

Sintaxis

  • Se basa en etiquetas.
  • Las etiquetas son las que definen el contenido de la página.
  • Las etiquetas se definen con los signos < y >.
  • Las etiquetas de apertura y cierre se definen con el signo /.
  • Usa una estructura jerárquica con elementos anidados.
  • Secciones principales: "<html>, <head>, <body>, <header>, <footer>, <section>".

Ejemplo de Código en HTML


<!DOCTYPE html>
<html lang="es"> // Documento HTML en español
  <head> 
    <meta charset="UTF-8">
    <title>Mi Página</title>
  </head>
  <body> //Cuerpo del documento
    <h1>Bienvenido</h1> //h1 es un encabezado
    <p>Parrafo.</p> p es un párrafo
  </body>
</html>Cierre del documento HTML
